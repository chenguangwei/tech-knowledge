# 规则引擎

DSL

领域特定语言，简称DSL，是专门针对特定应用领域的计算机语言，和可以用在多种领域的通用语言恰好相反。像HTML专门用在网页设计上，就属于领域特定语言

ANTLR

是基于LL算法实现的语法解析器生成器，用Java语言编写，使用自上而下的递归下降LL剖析器方法。由旧金山大学的Terence Parr博士等人于1989年开始发展
